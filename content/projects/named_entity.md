# Named Entity Standardization

LLM-powered system for standardizing inconsistent data entries using RAG and semantic similarity.

## Overview

A capstone project addressing the problem of inconsistent data entry in organizational databases. The system uses LLMs and Retrieval-Augmented Generation to standardize named entities automatically.

## Key Features

- **LLM-based standardization** using OpenAI API and RAG
- Deployed as an interactive **Streamlit web application**
- NLP pipeline with data cleaning and semantic similarity search
- Vector embeddings for finding similar entries
- Interactive visualizations for user review and approval

## Demo

Presented live demo to faculty and student panel, effectively communicating technical implementation and business value to non-technical stakeholders.
